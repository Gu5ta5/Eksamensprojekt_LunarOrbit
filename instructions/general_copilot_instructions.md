## Project Context: LunarOrbit
LunarOrbit is an interactive moon phase visualization tool. 
- **Core Focus:** Real-time astronomical data via API, visual moon rendering (analog and digital), and interactive time-scrubbing.
- **Secondary Focus:** A discrete system clock widget in the corner (system time).

3. **No Bloat:** Keep the UI minimalist. Focus on the moon visualization as the hero element.
4. **Error Handling:** Always include basic error handling for API requests and data parsing.
